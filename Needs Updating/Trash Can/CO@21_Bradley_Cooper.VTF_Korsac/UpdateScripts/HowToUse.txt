BACKUP YOUR FOLDER..

Place the UpdateScripts folder in the root folder of your missions. In the folder once in place, backup your mission folder before you do anything.. run the .bat file.

NOTE: The updater will delete some mission files (including Description.ext) so you can paste the latest template versions without overwrite prompts.

If it fails, or you notice something wasn't updated. Report it to me immediately and I will update the script to ensure that issue gets updated, could be anything, if its old name of codes, or whatever you find.

The more feedback the script gets, the better it will update our missions. Thanks! /Oksman

Once this is updated and done its thing, go into your root folder of the framework template and copy/paste the files it asks for (including Description.ext).

Then copy the import settings from the root folder of the framework template, import them into your addon options when you have loaded your mission in the addon options menu >> Import.
